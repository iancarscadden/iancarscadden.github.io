SMPL - A Simple, Educational Programming Language

Experience the joy of learning programming language concepts through a clean, minimal, and intuitive syntax. Perfect for beginners, educators, and language enthusiasts.

Download SMPL

Why SMPL?

Lightweight & Minimal

SMPL's syntax and core library are minimal, helping you focus on fundamental language concepts without overwhelming complexity.

Classes & Functions

SMPL supports object-oriented concepts like classes and methods, as well as user-defined functions, enabling structured code and reusability.

Control Flow & Loops

Master conditionals, loops, and logic with simple constructs like if, elif, else, for, and while.

Built-in Functions & Lists

Utilize built-in math functions and a powerful List class. Easily manipulate data and build on top of these primitives.

Clear Error Messages

Learn effortlessly with helpful error messages guiding you to understand and fix issues in your code.

Open Source & Educational

Fully open source. Explore the interpreter's code, experiment, and even contribute to its development on GitHub.

Getting Started

Setting up SMPL is straightforward and quick:

  1. Install Python 3.10+ & pip: Ensure Python and pip are available on your system.
  2. Clone the Repository: 
    git clone https://github.com/iancarscadden/smpl.git
  3. Install SMPL: 
    cd smpl
pip install .
  4. Run Your First Program: 
    smpl <filename.smpl>

After installation, verify by running:

smpl

You should see usage instructions if installed correctly.

Code Examples

Check out a few SMPL code snippets that highlight the simplicity and capabilities of this language:

Object-Oriented Example

class Person {
    name = ""
    age = ""
    height = ""

    func print_whole {
        print("Hello, my name is " + name +
              ". I am " + age + " years old." +
              " My height is " + height + ".")
    }
}

bob = new Person
bob.name = "Bob"
bob.age = "57"
bob.height = "5'7"
bob.print_whole()

class Animal {
    species = ""
    sound = ""

    func speak {
        print("The " + species + " says " + sound)
    }
}

dog = new Animal
dog.species = "dog"
dog.sound = "woof"
dog.speak()

cat = new Animal
cat.species = "cat"
cat.sound = "meow"
cat.speak()

Functions, Arithmetic & Conditionals

# Define a function to calculate the sum of two numbers
func add using a, b {
    return a + b
}

result_add = add(10, 20)
print("Addition Result: " + result_add)  # Should print 30

# Check if a number is even
func is_even using number {
    if number % 2 == 0 {
        return true
    } else {
        return false
    }
}

even_check = is_even(8)
print("Is 8 even? " + even_check)  # Should print true

Loops & User Input

// For loop
for i from 1 to 5 {
    print("Iteration: " + i)
}

// While loop
count = 1
while count <= 5 {
    print("Count: " + count)
    count = count + 1
}

// Get user input
print("What is your name?")
username = get_input()
print("Nice to meet you, " + username + "!")
