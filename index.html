About SMPL

SMPL is a new programming language that focuses on readability, maintainability, and extensibility. Designed with a clean syntax and robust parsing under the hood, SMPL makes it easier to write, debug, and scale your code.

Whether you're building small scripts or large systems, SMPL's recursive descent parser and AST-based architecture ensure that adding new features or understanding existing code is straightforward.

Key Features

  • AST-Based Parsing: Enjoy maintainable and easily extensible code, thanks to an internal architecture that uses an Abstract Syntax Tree for all expressions.
  • Clear Syntax: SMPL's syntax is designed to be intuitive and minimal, reducing cognitive load and making your code easier to review and understand.
  • Flexible Type System: Work seamlessly with numbers, strings, lists, and user-defined classes, all integrated into a cohesive environment.
  • Robust Error Handling: Detailed error messages help you quickly identify and fix issues in your code.
  • Built-In Functions & Extensibility: SMPL comes with common utilities (like `get_input`, math functions) and user-defined functions are easy to integrate.

Getting Started

To begin using SMPL, clone the official repository and run the interpreter on your first SMPL file. For example:


git clone https://github.com/username/smpl.git
cd smpl
python3 interpreter.py examples/hello_world.smpl

You can also define your own classes, functions, and use built-in methods from our standard library. Check out the documentation below for more details.
